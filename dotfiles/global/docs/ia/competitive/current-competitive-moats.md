## 📊 Visão Geral

## 🛡️ Moats Identificados

### 1. Sistema de Ranking por Mérito (Guardrail)

**Tipo**: Guardrail / Princípio

**Descrição**: Ranking baseado em dados verificáveis, não em pagamento

**Força**: 🟢 Forte

**Facilidade de Cópia**: 🟡 Média
- Requer mudança de modelo de negócio
- Concorrentes têm receita de mensalidades/destaque pago
- Mudar para ranking por mérito = perder receita

**Durabilidade**: 🟢 Alta
- Alinhado com valores fundamentais
- Dificilmente será abandonado
- Cresce em valor com o tempo

**Valor para Usuários**: 🟢 Alto
- Transparência total
- Igualdade de oportunidades
- Confiança no sistema

**Consultar**: `.cursor/docs/ia/guardrails.md`
