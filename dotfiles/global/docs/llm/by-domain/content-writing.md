## Recomendação por IDE

### Claude Code
- **Primary:** claude-sonnet-4-6
- **Razão:** Excelente redação em PT-BR técnico e criativo, tom natural e adaptável

### Cursor IDE
- **Primary:** claude-sonnet-4-6
- **Budget Cursor:** claude-haiku-4-5 (para textos curtos, variações de copy, títulos)
- **Quando Haiku:** Geração de variações de headline, CTAs, taglines, meta descriptions

### Codex (gpt-5.5)
- **Atual:** gpt-5.5 ✅ adequado para copy técnico
- **Upgrade recomendado:** — (gpt-5.5 é suficiente para maioria dos casos)
- **Migração sugerida:** → Claude Code para conteúdo longo com caching
