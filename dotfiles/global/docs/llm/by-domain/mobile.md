## Recomendação por IDE

### Claude Code
- **Primary:** claude-sonnet-4-6
- **Razão:** Excelente para desenvolvimento mobile cross-platform e nativo, bem treinado em Swift/Kotlin

### Cursor IDE
- **Primary:** claude-sonnet-4-6
- **Budget Cursor:** claude-haiku-4-5 (para componentes de UI simples, telas de formulário)
- **Quando Haiku:** Geração de telas CRUD, layouts básicos, configurações de build

### Codex (gpt-5.5)
- **Atual:** gpt-5.5 ✅ adequado para React Native e padrões mobile
- **Upgrade recomendado:** gemini-2.5-pro ($5.63/M) para integração Android/Google
- **Migração sugerida:** → Claude Code com Sonnet para arquitetura de apps complexos
