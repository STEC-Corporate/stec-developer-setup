# Planos (histórico)

Esta pasta guarda **planos aprovados ou em discussão** que a equipe quis versionar no repositório **AI-ProjectDeveloper** (montado como `.cursor/` no projeto pai).

- Os planos gerados pela UI do Cursor podem existir também em `~/.cursor/plans/` na máquina local — não substituem o registro aqui quando se pretende histórico no Git.
- Ao atualizar o submodule, fazer commit destes arquivos no repositório **AI-ProjectDeveloper**, não só no repo pai.
- Planos do produto **Olinket** (cortes C0–C9, PBR, suporte a termos, etc.) estão em [`Olinket/`](./Olinket/).
