# Example Skill

Esta é uma skill placeholder. Copie esta estrutura ao criar uma nova skill:

1. Pasta com nome em kebab-case
2. `SKILL.md` com frontmatter `name` e `description` e corpo com instruções
3. Opcionalmente: `reference.md`, `examples.md`, `scripts/`

Substitua o conteúdo pelo propósito real da sua skill.

## Quando usar
- Use quando a tarefa estiver alinhada ao objetivo da skill.
- Use quando for necessario padrao tecnico e repetibilidade.

## Pre-condicoes
- Escopo e contexto da tarefa definidos.
- Arquivos-alvo identificados.
- Criterios de aceite minimos estabelecidos.

## Checklist de saida
- Resultado entregue e verificavel.
- Evidencias objetivas registradas.
- Riscos e proximos passos documentados.

## Cobertura de stack e dominio
- Stacks suportadas: Java, Kotlin, Python, TypeScript, React, React Native, NextJS, NestJS.
- Dominios: backend, frontend e mobile.
