# Skill: Bases de dados e cache (Magalu Cloud)

Conteúdo **skeleton** — validar catálogo e regiões na [documentação oficial Magalu Cloud](https://magalu.cloud/) e comunicados de produto. Equivalências com hyperscalers são aproximadas.

## Quando usar

- Recursos e convenções do Magalu Cloud quando documentados para esta família.
- Cruzar com agents `cloud-aws-*` ou `cloud-gcp-*` para **equivalência conceptual** com hyperscalers.

## Checklist genérico

- [ ] Região e soberania de dados alinhadas ao requisito legal.
- [ ] Encriptação e controlo de acesso documentados pelo provedor.
- [ ] Plano de backup e recuperação para dados persistentes.

## Equivalência (referência externa)

Consultar catálogo cloud do provedor; comparar com família **datastores** em [catalogo-cloud-ia.md](../../../docs/governanca/catalogo-cloud-ia.md).
