# Skill: plugin-cursor-track-publication

## Quando usar

- Apos abrir PR de submissao.
- Durante ciclo de revisao e checks remotos.

## O que acompanha

- Status do PR (open/merged/closed).
- Checks do CI e falhas bloqueantes.
- Comentarios de revisao e itens pendentes.

## Saida

- Resumo de status atual.
- Pendencias priorizadas.
- Recomendacao final de publicacao.

## Script de apoio

- `scripts/track_publication.py`
