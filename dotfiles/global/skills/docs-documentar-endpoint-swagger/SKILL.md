# documentar-endpoint-swagger

## Documentos

Docs/CONSOLIDADO_FUNCIONALIDADES_FRONTEND.md, Docs/error-codes-reference.md, padrao Swagger do repo.

## Passos

1. Identificar endpoint.
2. Adicionar ou revisar ApiTags, ApiOperation, ApiResponse, exemplos.
3. Garantir DTOs com ApiProperty. Respostas de erro conforme error-codes-reference.
4. Manter consistencia com outros endpoints.

## Saida

Decorators Swagger completos. Documentacao alinhada ao contrato.
